# Calculator

一个纯 HTML + CSS + JS 实现的简单计算器，零依赖，支持键盘操作。

## 功能

- 四则运算（加减乘除）
- 小数点运算
- 链式计算（连续运算）
- 百分比（%）
- 退格（⌫）
- 全部清除（AC）
- 完整键盘支持

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `0-9` | 输入数字 |
| `.` | 小数点 |
| `+` `-` `*` `/` | 运算符 |
| `Enter` / `=` | 计算结果 |
| `Backspace` | 退格 |
| `Escape` / `Delete` | 全部清除 |
| `%` | 百分比 |
